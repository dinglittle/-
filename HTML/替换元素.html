替换元素:

1. 可替换 , 通过修改某个属性值程序的内容就可以被替换, image object video iframe

1) 内容的外观不受页面上的 CSS 影响.

2) 有自己的尺寸. video iframe canvas 默认 300 x 150, image 0 x 0

3) 在很多 CSS 属性上有自己的一套表现规则.

eg: select 是替换元素, 内容可替换; 基本样式外部 CSS 很难改变; 它有自己的尺寸;

2. 替换元素的默认 display 大部分 inline-block , Firefox 浏览器在 textarea 和 绝大多数类型的 input 元素上是 line 而不是 inline-block

input type="button" 和 button 按钮 前者 inline 后者 inline-block

3. 替换元素的尺寸元素计算规则
